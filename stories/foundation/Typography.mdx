# Typography

Font families are injected by the consuming project via `next/font` and exposed as CSS variables.
The library defaults to system fonts until those variables are set.

| Variable | Default |
|---|---|
| `--font-sans` | `system-ui, sans-serif` |
| `--font-mono` | `ui-monospace, monospace` |
